# ransomnote (WIP)
Makes letters look like they've been cut from newspaper.

## Features
- random font color;
- random background color.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```

### Tests
```
yarn run test:unit
```
